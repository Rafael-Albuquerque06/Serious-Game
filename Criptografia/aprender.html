Criptografias para aprender

Conceito de Chave : Uma chave física abre uma fechadura. Na criptografia, a chave digital "trava" (criptografa) a mensagem para que só quem tiver a chave certa possa "destrancá-la" (descriptografá-la).

Cifra de Cesar

Como funciona:

Cada letra do texto é substituída por outra letra que está um número fixo de posições (chave) mais adiante no alfabeto.

Exemplo (chave = 3) :

Texto original: A B C

Texto cifrado: D E F

Fórmula: Letra cifrada = (Letra original + chave) mod 26

Cifra de Deslocamento

Como funciona:

Cada caractere do texto é substituída por outro caractere que está um número fixo de posições (chave) mais adiante na tabela ASCII.

Exemplo (chave = 5) :

Texto original: HELLO 123

Texto cifrado: MJQQT 678

Cada caractere é deslocado 5 posições na tabela ASCII

Cifra de Substituição

Como funciona:

Cada letra é mapeada para outra letra de acordo com uma tabela de substituição fixa.

Exemplo de mapeamento (A chave é o mapeamento) :

A → X, B → M, C → P, ...
