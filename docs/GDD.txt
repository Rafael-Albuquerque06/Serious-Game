# Título do Jogo: CodeBreaker Mission

Identidade do jogo
CodeBreaker Mission é um serious game educativo onde o jogador assume o papel de um agente secreto que precisa descriptografar mensagens usando diferentes técnicas de criptografia para completar missões e salvar o mundo.

Descrição da mecânica
Jogo de puzzle educativo baseado em criptografia. O jogador recebe mensagens cifradas e deve aplica-las usando três tipos diferentes de cifras: Cifra de César, Cifra de Deslocamento ASCII e Cifra de Substituição (ROT13).

Características
- Ambiente de espionagem e missões secretas
- Sistema de pontuação e vidas
- Dicas contextuais para cada tipo de cifra
- Interface simples e intuitiva

Arte
Jogo 2D com design digital simples, seguindo o tema de terminal hacker com cores verdes sobre fundo preto.

Música/Trilha Sonora
Trilha sonora de suspense e espionagem, semelhante a filmes de agentes secretos.

Interface/Controles
Interface web responsiva com controles via teclado e mouse. O jogador digita respostas em campo de texto.

Dificuldade
- 3 níveis progressivos, cada um com um tipo diferente de cifra
- Sistema de vidas que penaliza respostas erradas
- Dicas disponíveis para auxiliar o jogador

Personagem
O jogador é um agente secreto anônimo trabalhando para uma organização de inteligência.

Cronograma
- Semana 1: Planejamento e design
- Semana 2: Implementação do código
- Semana 3: Testes e refinamentos
- Semana 4: Apresentação final

Definições gerais
- Gênero: Puzzle educativo
- Plataformas: Navegador web
- Quantidade de níveis: 3
- Público alvo: Estudantes de ciência da computação e interessados em criptografia
- Puzzles: Baseados em três tipos de cifras clássicas
